# 📝 homework — your solutions, class by class

Your own work for each assignment lives here, mirroring `uploads/`:

```
homework/
├── class-02-docker/    ← your Dockerfile, app.py, push-proof screenshot…
├── class-03-git/       ← your git-python-practice repo URL, git log --graph output…
└── class-NN-<topic>/
```

Suggested content per folder: the deliverables the assignment asks for + a short
`NOTES.md` with what tripped you up (future-you will thank you).

> [!warning] Sharing etiquette
> This repo is public so classmates can use the **study notes**. Homework
> *solutions* are a different story — don't push a solution before its deadline,
> and consider keeping graded work in a private branch if the course has
> academic-integrity rules. When in doubt, ask Yariv.

Practice runs for both current assignments exist in
[Shell Quest](https://github.com/iceteps/shell-quest) — missions 4 (Docker) and 5–7 (Git).
