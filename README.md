# 🎓 DevOps Experts — Study Vault

A complete, self-learning **Obsidian vault** for the DevOps Experts course:
one polished note per class, a master glossary, drills with XP, collapsible
self-quizzes, spaced-repetition flashcards, and the capstone project decoded.

> Companion game: **[🗡️ Shell Quest](https://github.com/iceteps/shell-quest)** —
> practice the commands from these notes in a simulated terminal (the Docker
> and Git missions mirror the course's real assignments).

## What's inside

| Note | What it gives you |
|---|---|
| `00 - Start Here (Setup and Plugins)` | 5-minute setup — **read this first** |
| `DevOps Experts — MOC` | the map: learning path, all notes, badge tracker |
| `Terminology` | master glossary — every note links into it |
| `DevOps Foundations` | the "why": Agile, CI/CD, deployment strategies |
| `Class 01–14` notes | Docker, Git, Kubernetes, Helm, GitOps, Ansible, Terraform, RabbitMQ |
| `SkyWatch Capstone` | the final project as a guided build roadmap |

Every class note follows the same self-learning flow:
**TL;DR → analogy → guided walkthrough → drills (XP) → learn-to-fish (find commands yourself)
→ self-quiz (collapsible) → flashcards → gotchas → boss challenge → collapsed cheat-sheet.**
The cheat-sheet is last and collapsed *on purpose* — attempt the drills before peeking.

## How to use it

1. **Get Obsidian** (free): https://obsidian.md
2. **Clone this repo** — either:
   - into your existing vault as a folder: `git clone https://github.com/iceteps/devops-study-vault "<your-vault>/devopsExperts"`, or
   - as a standalone vault: clone anywhere, then in Obsidian: *Open folder as vault*.
3. Open **`00 - Start Here (Setup and Plugins)`** and follow it (2 plugins, 5 minutes).
4. Optional but fun: install the **cheat-sheet flash** effect —
   copy `setup/cheatsheet-flash.css` into your vault's `.obsidian/snippets/` and enable it
   under *Settings → Appearance → CSS snippets*. Expanding a cheat-sheet then flashes a
   "you're peeking!" warning. 👀
5. Start at **`DevOps Experts — MOC`** and follow the learning path.

## Study loop (the short version)

Read the big idea → do the walkthrough → earn the drill XP → take the quiz →
review flashcards (Spaced Repetition plugin) → only then peek at the cheat-sheet.
Collect all 12 badges. 🎖️
